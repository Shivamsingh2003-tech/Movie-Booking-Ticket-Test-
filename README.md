# Movie Booking Ticket Test

A web-based **Movie Ticket Booking System** developed using **Java, JSP, Servlet, HTML, CSS, and Bootstrap**.
This project allows users to book movie tickets online through a simple and user-friendly interface.

## Features

* Movie ticket booking interface
* User-friendly homepage
* Ticket payment page
* Booking success page
* Error handling page
* JSP-based dynamic web pages
* Simple and clean UI
* Java web application structure

## Technologies Used

* Java
* JSP
* Servlet
* HTML
* CSS
* Bootstrap
* Apache Tomcat
* NetBeans / Eclipse / IntelliJ IDEA

## Project Pages

### Home Page

The home page allows users to view the movie booking interface and start the booking process.

### Payment Page

The payment page is used to enter payment-related details for ticket booking.

### Success Page

After successful booking, the user is redirected to the success page.

### Error Page

If any issue occurs during the booking process, the user is redirected to the error page.

## How to Run the Project

1. Clone the repository:

```bash
git clone https://github.com/Shivamsingh2003-tech/Movie-Booking-Ticket-Test-.git
```

2. Open the project in your Java IDE.

3. Configure Apache Tomcat Server.

4. Build and run the project.

5. Open the project in your browser using localhost.

Example:

```text
http://localhost:8080/Movie-Booking-Ticket-Test-
```

## Purpose of the Project

The purpose of this project is to demonstrate a basic online movie ticket booking system using Java web technologies. It is useful for learning JSP, Servlet-based page flow, and basic web application development.
